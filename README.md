# typescript-npm-stub

Features:

- Template for Typescript based node.js apps.
- Out of the box Logging.
- Out of the box App mechanics. 
- Includes test setup.
- Includes coverage setup.
- 100% coverage

To setup after cloning (or copying), run:
```bash
npm install
```

## How to's
- Compile:
```bash
npm run build
```
- Run App:
```bash
npm start
```
- Run Tests:
```bash
npm test
```
- Run Coverage:
```bash
npm run coverage
```
